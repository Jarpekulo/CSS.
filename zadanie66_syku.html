• pseudoelementem :first-line ---> Obejmuje formatowaniem cała pierwszą linie


    Przyklad:

    p
    {
    color:navy;
    font-size:12pt;
    }
    p:first-line
    {
    color:red;
    font-size:20pt;
    font-weight:bold;
    }

  • pseudoelementy: :after oraz :before ---> Oba elementy służą do umieszczania tekstu przed i za formatowanym elementem strony


    Przyklad:

    p:before
    {
    content:"TO SĄ ĆWICZENIA";
    font-size:15pt;
    font-style:italic;
    font-weight:bold;
    }
    p:after
    {
    content:"AUTOR Król Lew";
    font-size:15pt;
    font-style:italic;
    font-weight:bold;
    }


  • Pseudoelement :first-letter ---> pozwala na formatowanie pierwszej litery w bloku tekstu.

    Przyklad:

    p
    {
    color:navy;
    font-size:30pt;
    }
    p:first-letter
    {
    color:red;
    font-size:100pt;
    }
